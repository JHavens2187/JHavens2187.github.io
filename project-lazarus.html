Project: Lazarus

Retrofitting a 2013 iMac (Late 2013) to defy planned obsolescence using OpenCore Legacy Patcher and modern silicon.

HARDWARE / SOFTWARE

Phase 1: Software Injection

The first step was proving the architecture could handle the load. Using OpenCore Legacy Patcher (OCLP), I spoofed the SMBIOS to trick the kernel into accepting a modern OS.

  • Target: macOS Sonoma
  • Result: Successful Boot. Graphics acceleration patched.
  • Bottleneck: The 10-year-old 5400RPM HDD is throttling the system to a crawl.

Phase 2: The Transplant

To make the system usable, I must perform open-heart surgery. This involves cutting the display adhesive to access the internal SATA bus.

The Thermal Trap: Apple proprietary HDDs contain internal thermal firmware. Replacing the drive with a generic SSD causes the system fans to spin at 100% (Safety Mode).

The Fix: I will be installing an OWC In-line Digital Thermal Sensor to intercept the temperature data and normalize fan speeds.

Phase 3: Dual-Boot Architecture

Once the SSD provides the necessary IOPS, the final goal is a partitioned system running two distinct environments:

  • Partition A (Daily Driver): macOS Sonoma for development and UNIX-based workflows.
  • Partition B (The Console): Windows 11 for gaming support.
