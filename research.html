Constraining Dust Attenuation in AGN

Breaking the fundamental degeneracy between dust content and attenuation slope ($k(\lambda)$) in Active Galactic Nuclei using multi-line diagnostics.

INSTRUMENTATION & TOOLS
JWST / NIRSpec Python LMFIT Astropy GLEAM Pipeline

The Physics Problem

The standard method for estimating dust attenuation (the Balmer Decrement) is fundamentally degenerate. It can only solve for the normalization ($E(B-V)$) if the slope of the attenuation curve ($k(\lambda)$) is assumed, or vice versa.

The Solution: This research introduces a third constraint by combining Hydrogen Balmer lines ($H\alpha, H\beta$) with Paschen lines ($Pa\alpha, Pa\beta$) available via JWST. This breaks the degeneracy, allowing us to simultaneously derive both the dust content and the attenuation law specific to the complex AGN torus environment.

The Engine: GLEAM v2.0

I re-engineered the Galaxy Line Emission Analysis and Measurement (GLEAM) pipeline to handle large-scale spectral analysis for SMACS and JADES surveys.

@dataclass
class RandomVariable:
    """
    Custom class to handle error propagation for Multiplication and Division 
    during flux calculations. Solves the NaN issue in previous GLEAM versions.
    """
    value: Qty
    error: Qty

    def _propagate_multiplication_error(self, other):
        # Standard error propagation formula: |xy| * sqrt((dx/x)^2 + (dy/y)^2)
        if self.error is None or other.error is None:
            return np.nan
        
        val1 = self.value if self.value != 0 else 1e-9
        val2 = other.value if other.value != 0 else 1e-9

        relative_err_sq = (self.error / val1) ** 2 + (other.error / val2) ** 2
        return abs(self.value * other.value) * np.sqrt(relative_err_sq)
